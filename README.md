## Getting Started

1.Clone the repository:
https://github.com/LiveTicketGroup/ltg-livetixgroup-web-ui.git

2.Install the dependencies:
npm install
or
yarn install

3.Start the development server:
npm run start:dev
or
yarn dev

4.Launch http://localhost:3000

## Packages Used

- React Query (Data Fetching)
- Tailwind CSS
- Next SEO
- Eslint and Prettier
